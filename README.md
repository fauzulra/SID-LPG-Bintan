<h1>SID LPG Bintan</h1>

Sistem Informasi Distribusi Gas LPG 3 KG berbasis website atau bisa disebut sebagai SID LPG Bintan ini digunakan untuk kebutuhan Pemerintah Daerah (Dalam projek ini menggunakan studi kasus Pemerinta Kabupaten BIntan) dalam memantau persediaan stok subsidi gas LPG 3 KG serta memastika setiap penjualan subsidi LPG sesuai dengan aturannya.
